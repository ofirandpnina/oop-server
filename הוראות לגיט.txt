בתוך אמגולר כדי להגדיר גיט:
_________________________________________________________________________________________________________________________
git init   -- יוצר קובץ גיט
.  git add  ----  מוסיף לגיט המקומי את כל הקבצים שהיה בהם שינוי או שנוספו


בתוך האנגולר כדי להעלות את הקבצים :
_________________________________________________________________________________________________________________________
 git remote add origin https://github.com/ofirandpnina/oAndP   ----    התחברות לריפוסיטורי
git push -u origin master          -----------    בגיט המקומי הוספנו קבצים ועכשיו מוסיפים אותם לריפוסיטורי


הפעולות לשמירת שינויים
_______________________________________________________________________________________________________________________
git status  ----  כדי לראות את מצב הגיט
.  git add  ----  מוסיף לגיט המקומי את כל הקבצים שהיה בהם שינוי או שנוספו
git commit -m "change the style"  ---  שמירת השינוי והוספת כותרת
git push -u origin -----------    בגיט המקומי הוספנו קבצים ועכשיו מוסיפים אותם לריפוסיטורי



הורדת הפרויקט או השינויים למחשב
_______________________________________________________________________________________________________________________

git clone https://github.com/ofirandpnina/oop ---- הןרדת הפרויקט. לאחר ההורדה יש לבצע את הפקודה
npm i .


git pull https://github.com/ofirandpnina/oop  ---  הורדת השינויים בלבד. כגם כאן נטרך להתקין 
npm i .
